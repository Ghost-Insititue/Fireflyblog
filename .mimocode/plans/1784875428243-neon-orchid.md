# 编辑器页面导航栏视觉对齐

## 问题分析

编辑器页面的导航栏在视觉上与主页不同。根本原因：

**主页**：有侧边栏 → 浏览器有垂直滚动条 → 视口宽度 = `viewport - scrollbar_width`
**编辑器页**：隐藏侧边栏 + `overflow: hidden` → 无滚动条 → 视口宽度 = `viewport`

导航栏使用 `92vw` 宽度，vw 相对于视口宽度。因此：
- 主页导航栏宽度 = `0.92 × (viewport - scrollbar)`
- 编辑器页导航栏宽度 = `0.92 × viewport`（更宽）

当前的 `translateX(-8px)` 补偿是硬编码的，不适用于所有系统（滚动条宽度因 OS/浏览器而异）。

## 修改文件

| 文件 | 改动 |
|------|------|
| `src/pages/editor.astro` | 修复导航栏偏移计算，使用动态测量值 |
| `src/layouts/MainGridLayout.astro` | 移除硬编码的 `translateX(-8px)` |

## 具体改动

### 1. MainGridLayout.astro（~line 302）

移除服务端硬编码的偏移：
```diff
- style={isEditorPage ? 'transform: translateX(-8px)' : undefined}
+ style={undefined}
```
偏移完全由客户端动态计算处理。

### 2. editor.astro — 动态导航栏宽度补偿

**核心思路**：不使用 `92vw`（依赖视口），而是计算导航栏应该有的固定像素宽度，然后强制设置。

在 `applyEditorMode()` 中（~line 154），修改注入的 CSS：

```js
// 计算主页导航栏的有效像素宽度
// 主页视口 = window.innerWidth（有滚动条时）
// 编辑器页视口 = window.innerWidth（无滚动条，更宽）
// 导航栏在主页的有效宽度 = 0.92 × 主页有效视口
// 但 window.innerWidth 在两个页面上已经不同了

// 更好的方法：直接测量滚动条宽度，然后计算偏移
var scrollbarWidth = measureScrollbarWidth();
// 主页导航栏实际宽度 = 0.92 × (当前视口 - scrollbarWidth)
// 编辑器页导航栏宽度 = 0.92 × 当前视口
// 差值 = 0.92 × scrollbarWidth
// 需要左移的量 = 差值 / 2（因为导航栏是居中的）
// 但实际上导航栏是 w-full xl:w-[92vw]，不是居中的
// 所以需要的偏移 = 差值 = 0.92 × scrollbarWidth
```

修改 `editorStyleEl` 注入内容，将硬编码的 `translateX(-8px)` 替换为动态计算：

```js
var navbarOffset = Math.round(scrollbarWidth * 0.92);
// 注入 CSS 时使用计算值
'body.editor-page.sticky-navbar #top-row { transform: translateX(-' + navbarOffset + 'px) !important; transition: none !important; }'
```

### 3. editor.astro — resize 时重新计算

在 `editorResizeHandler`（~line 193）中，更新偏移量：

```js
var newSW = measureScrollbarWidth();
var newOffset = Math.round(newSW * 0.92);
rules = rules.replace(/translateX\(-\d+px\)/, 'translateX(-' + newOffset + 'px)');
```

### 4. editor.astro — 服务端渲染的初始偏移

在 `applyNavbarOffset()`（~line 125）中，也使用动态计算：

```js
function applyNavbarOffset() {
  var row = document.getElementById('top-row');
  if (row) {
    var sw = measureScrollbarWidth();
    var offset = Math.round(sw * 0.92);
    row.style.transform = 'translateX(-' + offset + 'px)';
  }
}
```

## 验证方式

1. `pnpm dev` 启动开发服务器
2. 在主页观察导航栏位置（左侧边缘距视口左边的距离）
3. 导航到 `/editor/`，观察导航栏位置
4. 两个页面的导航栏左侧边缘距视口左边的距离应相同
5. 缩放浏览器窗口，验证在不同宽度下都保持一致
6. 在不同浏览器/OS 上测试（滚动条宽度不同）
