text-sm / prose-sm 对应 14px，是代码编辑器的常见默认值（VS Code 默认也是 14px）。好处是屏幕上能显示更多内容，适合写作/编码场景。
如果觉得小了，可以调成：
- text-base / prose-base = 16px（浏览器正文标准大小）
- text-lg / prose-lg = 18px（偏大，阅读更舒适）
